3D Procedural Animation


About

This is a part of a project made for an assignment for RMIT University (Bachelor of Digital Media). The brief is to create a 3D procedural animation. I decided to create a portal underwater formation animation.


Process

Genshin Impact Portal
Finding Nemo

The animation took inspiration from the portal in Genshin Impact, an RPG game made by Mihoyo. The scene underwater inspiration is from Disney Finding Nemo.

Blender File 1
Blender File 2

I used geometry node to make the portal and its animation. I first created the round gate for the frame, then I used the point instance node and used the brick I made as the main object. The animation was created using attribute mix between position and the nodes attachment to create the formation of portal.
